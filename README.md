# health
Personal Health Data, Scripts, and Visualizations
This is the health data I have collected via my standard scale at home (for the weight measurement) and my health tracking watch (MorePro AIR2 Fitness Tracker) and my body fat composition scale (Xiaomi Mi Body Composition Scale 2).
I process the data using a python script periodically, which generates the attached visualizations so I can check to see how I'm doing.  Once a visualization is generated, the next line in the code won't be processed until the visualization is closed, and this continues until the last visualization is closed.
